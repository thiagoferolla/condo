# uCondo

Projeto desenvolvido para o desafio da vaga de Desenvolvedor Front-end da uCondo

## Tecnologias Utilizadas
- React Native
- Expo
- Sqlite
- Moti
- Dripsy
- Yup
- React Query

## Como Rodar

```
npm install && npm run android
```
